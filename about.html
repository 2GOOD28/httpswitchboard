Assets

HTTP Switchboard uses various assets which contents may change over time:

  • Lists of ubiquitous rules
  • Preset recipes
  • Public Suffix List

Since version 0.8.3.0, these assets can be updated here without having to wait for the next release of HTTP Switchboard. These assets are pulled from the project's Github repository.
