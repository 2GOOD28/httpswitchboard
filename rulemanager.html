Recipe

This is where you want to backup all of your scopes and rules: Click "Export rules", optionally click "Encode recipe", save the result into a file which content can be pasted here later.

Syntax is very strict, only the most geeky users should risk editing here. If you are importing a recipe from an untrusted source, be sure there is nothing wrong with it (ex.: “whitelist * evil.com”)

 A “recipe” is just a set of rules encoded in a way to make it easy to backup or exchange set of rules between users. Hopefully geekier users will share their carefully crafted recipes to help less geeky users.
